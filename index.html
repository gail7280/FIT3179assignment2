Summer Olympic Summary (1992 - 2008)

This visualization includes a range of idioms analyzing the medal statistics of summer Olympic games across 1992 to 2008.
The permanent members of the United Nations Security Council are chosen as sample data and referred as P5 for some idioms.


This map presents the total medal counts of all countries worldwide from 1992 to 2008 Olympic games.

Among all the countries, United State of America has won the most medals with a number of 1311. It is the only country with a total number over 1000.

Germany, Australia, Russia and China has also won impressive number of medals over 500.



This line chart shows the relationship between athlete numbers and medal numbers.

By selecting the country name in the drop-down menu, we can see the specific data regarding to these countries.

Overall, we can summaries that all the countries seems to have similar amount of medals.



This bump chart shows the rank change over time between P5.

By selecting the country name in the drop-down menu, we can see the trend regarding to only the selected country.

We can see that United States of America was always the top one country at the end of every summer Olympic game.



From the 3 donut charts we can see that the top 5 countries of winning different medals are very similar.

Within all the countries, United State of America has won much more medals than other countries.



This bar chart shows the number of participated males and females in each Olympic game.

By selecting the gender from the drop-down menu, we can filter out the data of only one gender.

Across 5 Olympic games, the number of participated male athletes are consistent, yet there is a increasing trend with the number of female athletes.
